# WebClient Studio — Sample Output Examples

This repo contains sample report output from [WebClient Studio](https://www.agensi.io/skills/webclient-studio) — a freelance web design pipeline skill bundle for AI agents.

All examples use the same fictional client (Summit Dental Care) to show how data flows between skills — the qualification research directly informs the proposal and onboarding brief.

## Samples

| File | Skill | What it shows |
|---|---|---|
| [lead-qualification-summ-dental-care.md](./lead-qualification-summ-dental-care.md) | Lead Qualifier | Full qualification report — website audit, tech stack analysis, scoring breakdown, findings, unverified items, recommendation |
| [proposal-summ-dental-care.md](./proposal-summ-dental-care.md) | Proposal Builder | Scoped proposal — executive summary, scope of work, timeline, pricing ranges, terms & conditions, next steps |
| [onboarding-project-brief.md](./onboarding-project-brief.md) | Project Onboarder | Project brief — client overview, goals, scope summary, contacts, technical requirements, content status, timeline |
| [onboarding-checklist.md](./onboarding-checklist.md) | Project Onboarder | Onboarding checklist — assets to collect, access & accounts, information needed, approvals process |

## How to read these

These are the **exact markdown files** the agent saves to disk during normal use. They live in `$WEBCLIENT_STUDIO_CONFIG_DIR/reports/clients/<company-slug>/` and can be viewed in any markdown viewer.

## Note

All company names, people, and data are fictional. Actual output will vary based on the company being researched and the agent's findings.
