# WebClient Studio — Sample Output Examples

This repo contains sample output from [WebClient Studio](https://www.agensi.io/skills/webclient-studio) — a freelance web design pipeline skill bundle for AI agents.

All examples use fictional companies. They demonstrate what the agent produces when running each skill.

## Samples

| File | Skill | What it shows |
|---|---|---|
| [lead-qualification-summ-dental-care.md](./lead-qualification-summ-dental-care.md) | Lead Qualifier | Full qualification report — website audit, scoring, fit assessment, findings, recommendation |
| [proposal-summ-dental-care.md](./proposal-summ-dental-care.md) | Proposal Builder | Scoped proposal — executive summary, scope, timeline, pricing, terms |
| [pipeline-view.md](./pipeline-view.md) | Pipeline Tracker | Master pipeline view — leads grouped by status with follow-up alerts |
| [csv-import-example.md](./csv-import-example.md) | Pipeline Tracker (§K) | CSV import chat flow — column mapping, preview, validation flags |

## How to read these

These are the **raw markdown outputs** the agent produces during normal use. They are saved to `$WEBCLIENT_STUDIO_CONFIG_DIR/reports/` and can be viewed in any markdown viewer.

The qualification report and proposal share the same fictional client (Summit Dental Care) to show how data flows between skills — the qualification research directly informs the proposal scope.

## Note

These are examples only. Actual output will vary based on the company being researched and the agent's findings. All company names, people, and data are fictional.
